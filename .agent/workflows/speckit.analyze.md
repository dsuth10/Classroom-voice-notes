# /speckit.analyze

Use this command to perform a project-wide audit against the constitution.

## Workflow

1.  **Audit Mode**: Check all code and documentation against `specs/constitution.md` and the feature designs in `specs/features/`.
2.  **Identify Violations**: List any deviations from principles, tech stack, Australian spelling conventions, or metric unit preferences.
3.  **Propose Fixes**: Provide actionable steps to align the project with its rules.
