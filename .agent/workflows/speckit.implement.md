# /speckit.implement

Use this command to begin implementing code according to the tasks defined in `specs/features/<feature-name>/tasks.md`.

## Workflow

1.  **Check Prereqs**: Ensure `specs/features/<feature-name>/spec.md`, `plan.md`, and `tasks.md` exist.
2.  **Iterate**: Execute tasks one by one.
3.  **Verify**: Run tests or manual checks after each significant task.
4.  **Update**: Mark tasks as completed in `specs/features/<feature-name>/tasks.md`.
